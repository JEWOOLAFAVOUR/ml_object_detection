# Sample Images Folder

Place your test images here for quick access in the Streamlit app.

## Supported Formats

- JPG, JPEG, PNG

## Recommended Test Images

- Photos with people (for person detection)
- Street scenes (for vehicle detection)
- Indoor scenes (for furniture detection)
- Food photos (for food item detection)
- Animal photos (for animal detection)

## Usage

1. Copy your test images to this folder
2. Run the Streamlit app: `streamlit run app.py`
3. Select images from the dropdown in the app

## Examples to Try

- Group photos with multiple people
- City streets with cars and buses
- Living rooms with chairs and TVs
- Kitchen scenes with utensils
- Outdoor scenes with animals

The app will automatically detect these images and make them available for testing.
